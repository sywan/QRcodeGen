# QR Studio Chrome Extension

Install this folder as an unpacked Chrome extension, or install the packaged zip in `release/`.

The toolbar button opens a compact QR Studio popup. Use **Open QR Studio** to launch the app in a Chrome extension tab.

## Install or reload

1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and choose this `chrome-extension` folder.
4. If QR Studio was already installed, click its **Reload** button on `chrome://extensions`.

## Show the toolbar button

Chrome may hide new extensions by default.

1. Click the puzzle-piece **Extensions** button in the Chrome toolbar.
2. Find **QR Studio**.
3. Click the pin icon beside it.
4. The QR Studio button should now stay visible in the toolbar.
